1. Ekstrak file ke folder htdocs
2. Buat database misal gallerymanager dan load file gallery-manager-sql.sql ke database tersebut
3. Ubah nilai variabel $base_url di app/config/config.php sesuai dengan lokasi dimana file php di ekstrak, misal jika file dikestrak di htdocs/gallerymanager, maka nilai $base_url diubah menjadi http://localhost/gallerymanager
4. Ubah konfigurasi database yang ada di file app/config/database.php
